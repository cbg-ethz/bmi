    bmi.samplers._tfp._product.ProductDistribution (JointDistribution)

    From distributions \(P_X\) and \(P_Y\) creates a distribution \(P_{XY} = P_X \otimes P_Y\), so that \(X\) and \(Y\) are independent.

    In particular, \(I(X; Y) = 0\).
